# IDK-interpreter
## Debugging
### Debug IDK programs 
```bash
DEBUG=1 ./idk.sh file.idk
```
### Debug the Interpreter
```bash
bash -xv idk.sh file.idk
```
### Or both
```bash
DEBUG=1 bash -xv idk.sh file.idk
```
## Usage
### stdin
```bash
./idk.sh
```
### file
```bash
./idk.sh file.idk
```

or

```bash
./idk.sh file.idk file1.idk file2.idk file3.idk ........
```
